# DecyAI Development Roadmap

## Product Position

DecyAI should not compete as another large AI tools directory. The winning angle is:

> Tell DecyAI what you need. It gives you the right AI stack, verified tools, exact prompts, and a practical workflow.

## Start Here

1. Validate one niche first: students, freelancers, small businesses, creators, or founders.
2. Keep recommendations free, but make advanced workflows and saved stacks paid.
3. Verify tools before showing them as top picks.
4. Publish useful pages for search traffic.

## Build Next

### 1. AI Stack Builder

Status: started.

Users should be able to ask:

```text
Build my AI stack for a student/freelancer/small business/creator/founder.
```

DecyAI should return a practical stack with:

- tool for research
- tool for writing
- tool for design
- tool for building
- tool for automation or publishing

### 2. Saved Stacks

Add accounts and allow users to save:

- Student Stack
- Creator Stack
- Startup Stack
- Business Marketing Stack

This is a strong subscription feature.

### 3. Verification System

Before charging users, verify each top tool for:

- current pricing
- free tier
- official website
- ideal use case
- limits
- best alternatives

Keep running:

```bash
npm run audit:tools
```

### 4. SEO Pages

Create pages like:

- `/best-ai-tools-for-students`
- `/ai-tools-for-small-business`
- `/best-ai-tools-for-logo-design`
- `/ai-stack-for-freelancers`
- `/ai-tools-for-startup-founders`

Do not write generic pages. Each page should include a ready-made stack and workflow.

### 5. Paid Plans

Start simple:

- Free: basic recommendations
- Pro: saved stacks, unlimited workflows, comparisons, prompt packs
- Team: shared approved stacks and business workflows

Suggested early pricing:

- India early users: Rs. 99/month
- Global early users: $3-5/month
- Later Pro: $7-12/month

## What Not To Do Yet

- Do not train your own AI model.
- Do not spend money on ads.
- Do not add hundreds of unverified tools.
- Do not build payments before at least 10 users say they want more limits/features.

## Weekly Execution Plan

### Week 1

- Test with 30 real users.
- Ask what task they used DecyAI for.
- Watch where recommendations feel wrong.
- Verify top 50 tools.

### Week 2

- Add saved stacks.
- Add 5 SEO pages.
- Add better comparison pages.

### Week 3

- Add login.
- Add usage limits.
- Add simple Pro gate.

### Week 4

- Launch publicly.
- Post 1 demo every day.
- Collect first paying users manually if needed.
